## Metrics to Track

**Key Performance Indicators (KPIs):**

1. **Conversion Rate** - Landing page → Assessment purchase
   - Target: 3-5%

2. **Cost Per Assessment** - Marketing spend ÷ Assessments
   - Target: <$200

3. **Upsell Rate** - Assessments that upgrade to implementation
   - Target: 40-60%

4. **Implementation Value** - Average upsell amount
   - Target: $4,000-5,000

5. **Customer Acquisition Cost (CAC)**
   - Formula: Marketing spend ÷ Total customers
   - Target: <$400

6. **Lifetime Value (LTV)**
   - Assessment + Upsell + Retainer value
   - Target: >$5,000

7. **LTV:CAC Ratio**
   - If LTV = $5,000 and CAC = $400, ratio = 12.5:1
   - Target: >3:1
